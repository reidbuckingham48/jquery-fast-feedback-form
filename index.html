Fast Feedback Form

Objective

This project is designed to enhance user experience through interactivity that provides feedback when the user inserts correct/incorrect information.

When the user enters incorrect information, the text box that is selected will turn red and a prompt will be given as to what criteria must be met. Once correct information is entered, the box will turn green.

If the user does make an error, please refresh the page and start over (this is a bug I am still trying to figure out). If correct input is put in, the color around the box will change but the error notification will not clear.
